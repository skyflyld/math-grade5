### 1. 🧱 拆到公理

```
【第1屏】⚡ 10秒闪电提问
"乘法到底是什么？"
别往下翻。先想3秒。

[3秒后自动显示]
3 × 4 = 4 + 4 + 4
3 × 4 = 3个4相加

所以，乘法的本质是——
求几个相同加数的和。
```

```
【第2屏】追着打
"那 3 × 0.4 呢？"

[学生思考3秒后显示]
还是3个0.4相加。
0.4 + 0.4 + 0.4 = 1.2

乘法没变。
变了的是：0.4的计数单位是0.1（十分之一），
4的计数单位是1。

3 × 4：  3个"1" × 4  = 12个"1"
3 × 0.4：3个"0.1" × 4 = 12个"0.1" = 1.2

公理：
━━━━━━━━━━━━━━━━━
乘法的本质没变。
变的只是"计数单位"。
学会换单位，小数乘法 = 整数乘法。
━━━━━━━━━━━━━━━━━
```

```
【第3屏】一句话总结（费曼式，学生念出来）
"小数乘法就是把小数当整数算，
　然后再把计数单位换回来。
　就这么简单。"
```

### 5. ⚠️ 踩坑预警

**坑①：竖式对齐 —— "末位对齐" vs "小数点对齐"**

```
【症状展示】
同学小明的竖式：
   9.5
×   3
─────
 28.5        ← 正确（末位对齐）

同学小红的竖式：
   9.5
×   3
─────
  285         ← 小数点和3对齐了！错！

小红想："加法要小数点对齐，
　　　　乘法也应该小数点对齐吧？"
```

```
【根因解析】
加法要求"相同数位对齐"（个位对个位、十分位对十分位），
所以小数点必须对齐。

但乘法不是在同位相加——是全部相乘再合并。
所以乘法用"末位对齐"。

混淆的根源：把加法的规则套到了乘法上。
```

```
【后续炸雷】
如果这里用"小数点对齐"，到了小数除法中——
列竖式时也会把小数点和被除数对齐而不是按除数转化。
一步错，后面的小数除法全错。
```

```
【自查法】
写完竖式后问自己：
"我在做加法还是乘法？"
加法 → 小数点对齐
乘法 → 末位对齐，之后单独处理小数点
```

```
【纠正练习】
判断下面的竖式哪个对，哪个错：
①   4.6        ②   4.6
  ×  3            ×   3
  ─────           ─────
   13.8             138
答案：①对②错。②忘了点小数点。
```

**坑②：小数位数死记硬背**

```
【症状展示】
老师教了："积的小数位数 = 因数小数位数之和"
学生背了，然后做这道题：
0.4 × 0.5 = 0.20 → 化简 = 0.2

学生慌了："0.4一位，0.5一位，加起来两位。
　　　　0.2只有一位？我算错了？"

他没算错。0.20=0.2。是口诀制造了焦虑。
```

```
【根因解析】
"积的小数位数等于因数小数位数之和"——这条规则正确，
但学生不明白为什么，变成了机械口诀。

正确的理解：
0.4是4个0.1，0.5是5个0.1
4×5=20（个0.1×0.1=0.01）
20个0.01 = 0.20 = 0.2

不是"数位数"，是"理解每次乘法的单位变化"。
```

```
【后续炸雷】
死记口诀到分数乘法彻底崩盘。
2/3 × 1/4 = 2/12，背的是"分子乘分子、分母乘分母"，
而不理解"2/3的一半 = 1/3"。
口决算不出理解，分数是试金石。
```

```
【自查法】
每次做完小数乘法，用换单位法重做一次：
0.4×0.5
→ (4个0.1)×(5个0.1)
→ 20个(0.1×0.1)
→ 20个0.01
→ 0.20 = 0.2 ✓

如果换单位法算出来和口诀不一样 → 优先相信换单位法。
```

### 0. 前置门控
复习"小数×整数"的核心：0.6×7=（换单位法验证）

### 1. 🧱 拆到公理
起手式："刚才我们只会小数×整数。如果两个都是小数呢？"
公理：**"两个小数相乘 = 两个整数相乘 ÷ 单位缩小倍数的乘积"**
0.4×0.5 = (4×5) ÷ (10×10) = 20÷100 = 0.2

### 2. 🎮 交互实验室
- 面积模型：长0.8宽0.4的矩形→分成100格→32格=0.32
- 小数点联动拖动：改因数的小数位数→观察积的位数自动变化

### 3. 📐 教材溯源
"长2.4m宽0.8m的宣传栏刷漆"→教材为什么选面积？因为面积天然是"乘法"的视觉化。

### 4. 🎯 典例精讲
- 模式识别：0.56×0.04（位数不够用0补足）
- 验证：给出0.48×1.5=0.72，判断对错并解释原因

- 坑③：位数不够时忘记用0补足（"0.06×0.05=0.003→正确，但我写成了0.03"）
