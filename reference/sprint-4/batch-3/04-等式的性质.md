# 等式的性质 · 教学参考

> 课件：`modules/05-equations/5-2-equality-properties.html`  
> 共享组件：`createBalanceScale`  
> data-concept：等式的性质,简易方程

## 公理

1. **等式两边加减同一个数，等式仍成立**：如果 a=b，则 a+c=b+c，a-c=b-c。这是方程变形的基础。
2. **等式两边乘除同一个非零数，等式仍成立**：a=b → a×c=b×c (c≠0)；a÷c=b÷c (c≠0)。
3. **等式可传递**：a=b 且 b=c → a=c。
4. **天平是等式性质的物理模型**：两边重量相等时平衡，同时加减相同重量仍平衡。

## 常见误区

| 误区 | 症状 | 根因 | 自检链 |
|------|------|------|--------|
| 只变一边 | 解 x+5=12，只在左边减 5 | 忘记了等号两边是整体 | 左边减了 5，右边减了吗？ |
| 除 0 的陷阱 | 两边同时除以 (x-3)，忘记 x-3 可能为 0 | 没检查除数是否为零 | x=3 时 x-3=0，不能除。 |
| 乘负数忘变号 | 不等式中乘负数要变号，等式中不混淆 | 把不等式规则带入等式 | 等式两边乘负数，等号方向不变。 |

## 对抗挑战（4 道）

1. **tag: 两边同变**  
   statement: 解方程 x-7=15，只需要把左边的 -7 变成 +7 就可以了。  
   answer: refute  
   hint: 给左边 +7 后，右边呢？  
   explanation: 等式两边必须同加 7：x-7+7=15+7，x=22。  
   counterexample: 如果只给左边加 7，左边变成 x，右边还是 15，等号不成立。

2. **tag: 除零陷阱**  
   statement: 方程 3x=15，两边可以同时除以任何数。  
   answer: refute  
   hint: 能除以 0 吗？  
   explanation: 除法要求除数不为 0。3x=15 可以除以 3，但不能除以 0。  
   counterexample: 除以 0 没有数学意义。

3. **tag: 乘除为零**  
   statement: 如果 a=b，两边同时乘以 0，得到 0=0，所以方程的解还是原来的。  
   answer: refute  
   hint: 从 0=0 能推出 a=b 吗？  
   explanation: 乘 0 是不可逆操作。乘 0 后信息丢失，不能推回原式。  
   counterexample: 方程 x=5，两边乘 0 得 0=0，但 0=0 对应的 x 不一定是 5。

4. **tag: 逆向思考**  
   statement: 4x+2=18，用逆运算解：先减 2 再除 4。  
   answer: agree  
   hint: 逆运算顺序和正序相反。  
   explanation: 正序：x→×4→+2。逆序：18→-2→÷4→15→÷4→... 18-2=16，16÷4=4。x=4。  
   counterexample: 验算：4×4+2=18。

## 分层练习

### 组1：直接应用
1. x+8=20，x=？（答案：12）
2. 3x=18，x=？（答案：6）
3. x-5=9，x=？（答案：14）

### 组2：变式迁移
1. 2x+3=15，x=？（答案：6）
2. 5(x-2)=20，x=？（答案：6）
3. 等式 a=b 两边同时除以 c，需要什么条件？（答案：c≠0）

### 组3：综合应用
1. 为什么等式两边乘 0 会丢失信息？（答案：0=0 无法推出原等式）
2. 用等式的性质解释"移项要变号"。（答案：移项=两边同加减，右移到左变号）
3. 3(x+2)-x=2x+10，x=？第几步用了哪个性质？（答案：x=4）
