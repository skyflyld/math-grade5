# 真分数假分数 · 教学参考

> 课件：`modules/14-fraction-meaning/14-4-proper-improper.html`  
> 共享组件：`createNumberLine`（分数数轴分类模式）  
> data-concept：真分数和假分数,分数的意义

## 公理

1. **真分数 < 1，假分数 ≥ 1**：真分数分子<分母（如 3/4），假分数分子≥分母（如 5/4）。这是纯粹大小关系，不是"好/坏"。
2. **假分数不是"错的分数"**：它只是恰好分子≥分母。5/4 在数轴上比 1 大，完全合法。
3. **带分数 = 整数 + 真分数**：1¾ = 1 + 3/4 = 7/4。假分数 ←→ 带分数可以互化。
4. **互化规则**：假分数→带分数：分子÷分母，商是整数部分，余数是分子。带分数→假分数：整数×分母+分子。

## 常见误区

| 误区 | 症状 | 根因 | 自检链 |
|------|------|------|--------|
| "假分数是不对的" | 拒绝写 7/4，一定要写 1¾ | "假"字的字面含义被当成了数学含义 | 7/4 和 1¾ 在数轴上同一点吗？ |
| 互化算错整数 | 9/4=1 又 5/4 | 没把整数部分取最大 | 5/4 还能再分出整数吗？ |
| 大小判断错 | 3/4 > 5/4 因为 3<5 但分母相同 | 只看分子，忘了看是不是真分数 | 3/4<1<5/4，一目了然。 |
| 带分数加法 | 1¾+2½ = 3¾+½... | 没先化为同分母 | 先通分再算。 |

## 对抗挑战（4 道）

1. **tag: 定义辨析**  
   statement: 7/7 是一个真分数，因为分子等于分母。  
   answer: refute  
   hint: 真分数的条件是什么？  
   explanation: 真分数要求分子严格小于分母。7/7=1，是假分数。  
   counterexample: 2/3 是真分数，因为 2<3。

2. **tag: 大小判断**  
   statement: 真分数都小于 1，假分数都大于 1。  
   answer: refute  
   hint: 3/3 是假分数，它等于多少？  
   explanation: 假分数 ≥ 1。分子=分母时，假分数 = 1。  
   counterexample: 3/3=1，5/5=1，都是假分数但等于 1。

3. **tag: 互化验证**  
   statement: 11/4 = 2¾，因为 11÷4=2 余 3。  
   answer: agree  
   hint: 验算：2¾ = 2×4+3/4？  
   explanation: 2×4+3=11，所以 2¾ = 11/4。互化正确。  
   counterexample: 用加法验证：2+3/4=2.75，11/4=2.75。

4. **tag: 逆向思考**  
   statement: 一个假分数化成带分数是 3 又 2/5，原分数是 17/5。  
   answer: agree  
   hint: 3×5+2=?  
   explanation: 3×5+2=17，分子是 17，原分数 17/5。  
   counterexample: 17/5÷5=3 余 2，互化回去也是 3¾...不是，是 3⅖。

## 分层练习

### 组1：直接应用
1. 判断真/假：3/5（ ）、7/4（ ）、9/9（ ）。(答案：真、假、假)
2. 8/3 化成带分数是（ ）。(答案：2⅔)
3. 2¾ 化成假分数是（ ）。(答案：11/4)

### 组2：变式迁移
1. 把 2⅔ 和 8/3 在数轴上标出来，它们的位置关系是？（答案：同一点）
2. 从小到大排：5/6、1¼、2/3、3/2。(答案：2/3 < 5/6 < 1¼ < 3/2)
3. 妈妈把 3 个同样大的披萨平均分给 4 个小朋友，每人分到多少？（用两种写法）(答案：3/4 个 或 都可以)

### 组3：综合应用
1. 为什么做分数加减时常把带分数化为假分数？(答案：便于通分和统一计算)
2. 写出 3 个最简真分数，使它们的分母都是 8。(答案：1/8, 3/8, 5/8, 7/8)
3. 小明说"假分数就是错的分数，考试不能写"，你怎么解释？(答案：假分数是数学中完全合法的表示)
