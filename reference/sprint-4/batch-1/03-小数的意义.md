## 公理

1. **小数是十进制的自然延伸**：整数向右每移一位缩小 10 倍——个位→十分位→百分位→千分位。只是方向反了，逻辑完全一致。
2. **小数点只是"个位锚"**：它标记了个位的位置。左边 10 倍扩展，右边 1/10 缩微。
3. **0 在小数中不等于"无"**：0.5 和 0.50 等值，但 0.05≠0.5。0 的位置决定值的大小。
4. **小数计数单位**：十分位计数单位是 0.1，百分位计数单位是 0.01。和整数的"个十百千"是同构的。

## 常见误区

| 误区 | 症状 | 根因 | 自检链 |
|------|------|------|--------|
| 小数越大越"长" | 0.123 > 0.8 | 按整数经验，位数多=大 | 对齐小数点后逐位比较。 |
| 末尾 0 盲目去掉 | 2.50 = 2.5 是对的，但结论"0 不重要" | 没区分"末尾0"和"中间0" | 2.05 去掉 0 成 2.5？不可能。 |
| 小数点位置随意 | 3.14 读作"三百一十四" | 忽略了小数点标记的位值 | 小数点左边再读位名。 |
| 换算方向反 | 0.3 米 = 3 厘米 | 进率 100 时方向判断错 | 大单位→小单位，乘进率。 |

## 对抗挑战（4 道）

1. **tag: 位数陷阱**  
   statement: 0.37 比 0.9 大，因为 37>9。  
   answer: refute  
   hint: 小数点对齐后，十分位谁大？  
   explanation: 十分位 3 < 9，所以 0.37 < 0.9。不能只比数字不看位值。  
   counterexample: 0.9 元是 9 角，0.37 元是 3 角 7 分，9 角更多。

2. **tag: 0 的区分**  
   statement: 2.5 = 2.50 = 2.500，所以小数的 0 可以随便加减。  
   answer: refute  
   hint: 2.05 去掉 0 变成多少？  
   explanation: 只有末尾的 0 可以去掉。中间的 0 改变位值。  
   counterexample: 2.05 去掉 0 变成 2.5，值变了 50 倍。

3. **tag: 单位换算**  
   statement: 0.35 千米 = 35 米。  
   answer: refute  
   hint: 1 千米 = ? 米  
   explanation: 0.35km = 0.35×1000 = 350m。进率是 1000。  
   counterexample: 0.35×1000=350，不是 35。

4. **tag: 逆向思考**  
   statement: 一个数由 3 个一、4 个 0.1 和 5 个 0.01 组成，这个数是 3.45。  
   answer: agree  
   hint: 3×1 + 4×0.1 + 5×0.01。  
   explanation: 个位 3、十分位 4、百分位 5，写成 3.45。  
   counterexample: 位值表：个个十百分，3.4.5 → 3.45。

## 分层练习

### 组1：直接应用
1. 0.6 里面有（ ）个 0.1。(答案：6)
2. 3.25 的百分位上的数字是（ ）。(答案：5)
3. 比较：0.78 ○ 0.8（填 > < =）。(答案：<)

### 组2：变式迁移
1. 把 5 个 0.1、3 个 0.01 和 2 个 0.001 写成小数。(答案：0.532)
2. 0.45 米 =（ ）厘米。(答案：45)
3. 3.05 和 3.50 哪个大？(答案：3.50)

### 组3：综合应用
1. 用数字 2、3、0、5 组成最大的两位小数。(答案：53.20)
2. 小明说 0.8=0.80，但老师说 0.80 更精确，为什么？(答案：0.80 精确到百分位)
3. 在数轴上标出 0.25、0.5、0.75 的位置并说明它们之间有什么关系。(答案：每段差 0.25)
