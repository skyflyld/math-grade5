# 约分 — 教学参考

## 拆到公理

**核心**：约分是**通分的逆操作**——把分数等份"合并"让它更简洁，但值不变。本质仍然是分数的基本性质：a/b = (a÷k)/(b÷k)，其中 k 是 a 和 b 的公因数。

**不可再拆的公理**：
1. 分数的基本性质同样适用于除法方向（分子分母同除一个非0公因数）
2. 最简分数 = 分子分母互质时（只有公因数1）
3. 约分不改变分数值，只改变"表示方式"

## 前置依赖

- 分数的基本性质（与通分共用同一根基）
- 因数和倍数
- 公因数和最大公因数（GCF）
- 通分的概念（作为对比参考）

## 常见误区 Top 3

1. **约分约不彻底**：用 2 约了一次就停了，没注意到还能用 3 再约一次。
   - 例：12/18 → 6/9（用2约）→ 停住了，其实还能用3约到 2/3
2. **错把减法当约分**：认为"3/6 约分就是分子分母各减1得 2/5"——完全搞错操作。
3. **对"为什么能约分"不理解**：机械操作，不理解"等份合并"的几何意义。

## 交互实验室方案

**fraction-bar 组件**（约分方向）：
- 初始显示 12/18（18等份取12份，蓝色）
- 点击"找公因数 2" → 每2个小格合并成1大格 → 9等份取6份 → 6/9
- 点击"找公因数 3" → 每3个小格合并成1大格 → 3等份取2份 → 2/3
- 点击"一步到最简" → 找GCF=6，每6格合并 → 直接到 2/3
- 关键：蓝色总面积**始终不变**，让学生看到约分前后的等值性
- 附加滑块：手动输入公因数，尝试不同约分路径

## 对抗挑战 4 道

1. **小华的答案**：把 24/36 约分，小华写 24/36 = 12/18。小刚说"还可以继续约"。小华不服："我已经约过了！"谁对？24/36 的最简分数是什么？

2. **找反例**："分子分母都是偶数就约完了。"这句话对吗？12/30 用这句话处理会出什么问题？

3. **识别陷阱**：8/12 = ?/3。小红写 2/3，小蓝写 4/6。谁的答案最简洁？4/6 还能再约吗？

4. **逆向思考**：一个最简分数，分子是 3，分母在 10 到 20 之间。这个分数可能是哪些？
