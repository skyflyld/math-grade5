# 分数加减法 — 教学参考

## 拆到公理

**核心**：分数加减法与整数加减法基于**同一个公理**——只有**相同单位的量**才能直接加减。整数加减法对齐数位（个位对个位、十位对十位），分数加减法对齐分数单位（分母相同 = 等份大小相同）。

**不可再拆的公理**：
1. 分数单位：分母决定"每一份多大"，同分母 = 同单位
2. 同分母加减：分子加减，分母不变（单位相同，只改变份数）
3. 异分母加减 = 先通分（统一单位）→ 再同分母加减
4. 结果能约分的要约分（保持最简）

## 前置依赖

- 分数的意义（分母=等份数，分子=取几份）

## 常见误区 Top 3

1. **分母直接相加**：1/3 + 1/3 = 2/6（学生把分母也加了）——这是最经典的错误，根源在于不理解"分母=单位大小"。
2. **只通分不加约简**：异分母加减后，结果不约分。如 2/6 + 1/6 = 3/6，停在 3/6 不约到 1/2。
3. **带分数进退位混乱**：整数部分和分数部分各自进退位不会处理。

## 交互实验室方案

**fraction-bar 组件**（加减模式）：
- 两条平行的分数条：上方红色 1/3，下方蓝色 2/3
  - 点击"相加" → 蓝色条拼接到红色条后面 → 总长 3/3 = 1 → 完整条变绿
- 上方 2/5，下方 1/3（异分母）
  - 点击"通分" → 两条同时切到 15 等份 → 6/15 和 5/15
  - 点击"相加" → 拼接 → 11/15
- 减法模式：上方 5/6，下方拖走 1/3
  - 先通分到 5/6 和 2/6 → 从红色条中"切掉"2格蓝色 → 剩余 3/6 = 1/2
- 关键：加减操作在**同一刻度尺**上进行，学生看到"单位统一"后才发生加减

## 对抗挑战 4 道

1. **小明的计算**：1/4 + 1/4 = 2/8。小华说："不对，1块钱+1块钱=2块钱，不是2分钱。分母是单位大小，不能加。"小明不服："那为什么整数加法个位和个位加？"小华该怎么解释？

2. **找反例**："分数加法就是分子加分子，分母加分母。"举一个例子让说这句话的人立刻意识到自己错了。

3. **识别陷阱**：计算 3/8 + 1/4。小红通分后写成 3/8 + 2/8 = 5/8。小蓝说"答案是 5/16，因为 8+8=16"。谁错了？小蓝错在哪？

4. **逆向思考**：两个分数相加等于 5/6，其中一个分数是 1/2。另一个分数是多少？写出你的推理。

## 分层练习 3 组

**组1 直接应用**：同分母：3/7 + 2/7；5/9 - 2/9。异分母：1/4 + 1/6；5/8 - 1/3。

**组2 变式迁移**：不计算，估计 7/12 + 5/12 的结果是大于 1 还是小于 1？如何快速判断？3/10 + 2/5 和 2/5 + 3/10 结果一样吗？为什么？
