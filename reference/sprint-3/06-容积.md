# 容积 — 教学参考

## 拆到公理

**核心**：容积 = 容器内部能容纳物体的**体积**。容积和体积的计算公式**完全相同**（长方体：长×宽×高），唯一的区别是：体积量的是物体本身占的空间，容积量的是物体**内部空腔**能装多少东西。

**不可再拆的公理**：
1. 容积的计算公式 = 体积的计算公式
2. 容积与体积的区别：体积 = 外部尺寸，容积 = 内部尺寸（壁厚导致差异）
3. 容积单位：升(L)和毫升(mL)，1L = 1dm³ = 1000cm³，1mL = 1cm³
4. 对于薄壁容器，容积 ≈ 体积（教学中通常这样简化）

## 前置依赖

- 长方体和正方体
- 体积和体积单位
- 长方体体积 = 长×宽×高
- 单位换算（cm³ ↔ mL ↔ L）

## 常见误区 Top 3

1. **混淆容积和体积**：以为"容积就是体积"，不理解内部空间和外部占位的区别。
2. **单位混淆**：1L = 1000mL，但 1m³ ≠ 1000cm³（实际是1,000,000cm³），学生容易把千进制搞混。
3. **壁厚忽略**：计算鱼缸容积时直接用外部长×宽×高，忘记减去玻璃厚度。

## 交互实验室方案

**Canvas 注水动画**：
- 3D 透视长方体容器（标注内部尺寸：长30cm、宽20cm、高25cm）
- 点击"注水" → 水面从底部升起，动画展示水位上升
  - 水面到 5cm：显示 30×20×5 = 3000cm³ = 3L
  - 水面到 15cm：显示 30×20×15 = 9000cm³ = 9L
  - 水面到 25cm：显示 30×20×25 = 15000cm³ = 15L
- 切换对比：同一容器，把水的体积倒入量杯 → 显示对应的升数
- 关键：让学生看到**水的形状 = 容器的内部形状**，水位高度 × 底面积 = 水的体积 = 容器这部分空间的大小

## 对抗挑战 4 道

1. **小明的判断**："一个纸箱的体积是 60dm³，所以它的容积也是 60dm³。"这句话对吗？什么时候体积和容积差不多？什么时候差很多？

2. **找反例**："容积的单位就是升和毫升。"这个说法有什么问题？1cm³ 能不能表示容积？

3. **识别陷阱**：一个长方体鱼缸，外部尺寸：长50cm、宽30cm、高40cm，玻璃厚1cm。鱼缸的容积是多少？小红算 50×30×40=60000cm³=60L。她算对了吗？

4. **逆向思考**：一个容器的容积是 8L，底面积是 2000cm²。这个容器有多高？

## 费曼填空 4 道

1. "容积和体积的计算公式一样，但量的东西不同：体积量的是______________________，容积量的是______________________。"
2. "1升水倒进一个 1dm³ 的正方体盒子里，______________________（刚好装满/装不满/溢出），因为______________________。"
3. "计算有厚度的容器容积时，要用______________________尺寸，而不是______________________尺寸，因为______________________。"
4. "升和毫升是特殊的体积单位，1L = ____dm³，1mL = ____cm³。它们的好处是________________________________。"

## 分层练习 3 组

**组1 直接应用**：一个长方体容器内部尺寸：长40cm、宽25cm、高30cm，容积是多少升？

**组2 变式迁移**：一个鱼缸外部尺寸50cm×30cm×40cm，玻璃厚1cm。装满水后，水的体积是多少升？如果把水倒进一个底面积1000cm²的圆柱形容器，水面多高？

**组3 综合应用**：一个蓄水池内部尺寸：长8m、宽5m、深2m。现在池内有32m³的水。再注入多少立方米水会溢出？如果每分钟注入2m³，还需要多少分钟？
