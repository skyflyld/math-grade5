# 循环小数 — 教学参考

## 拆到公理

**核心**：循环小数的本质是**除法中余数的周期性重复**。当一个除法算式的小数部分除不尽时（不能写成有限小数），余数会周期性地重复出现，导致商的数字也周期性重复。这不是"数字碰巧循环"，而是**有限个可能余数（分母-1个）必然导致重复**。

**不可再拆的公理**：
1. 除法 = 不断用余数×10继续除下一步
2. 除以n时，余数只能是0到n-1中的某个数
3. 如果除法不结束（商不是有限小数），n-1步内余数必然重复（鸽巢原理）
4. 余数重复 → 商的数字重复 → 形成循环节

## 常见误区 Top 3

1. **以为"循环"是巧合**：学生不理解循环的必然性，以为"刚好除出来就这样"。
2. **循环节识别错误**：0.333...的循环节是3，但学生可能写成0.3̇3̇（把两个3都标循环点）。
3. **混淆有限小数和循环小数**：1/4=0.25是有限小数，1/3=0.333...是循环小数。学生可能以为"只要除不尽就是循环"。

## 交互实验室方案

**long-division 组件**（循环小数模式）：
- 竖式除法：1÷3
  - 第1步：1÷3=0余1 → 商0，余1
  - 第2步：10÷3=3余1 → 商0.3，余1
  - 第3步：10÷3=3余1 → 商0.33，余1
  - 第4步：余数1再次出现！高亮循环
  - 自动标出循环节：0.3̇
- 竖式除法：2÷7
  - 展示完整的6位循环节（142857）
  - 提示：除以7，余数最多6种可能，所以循环节最长6位
- 关键：**余数高亮**是核心——学生看到同一个余数出现两次，就理解了循环的必然性

## 对抗挑战 4 道

1. **小明的断言**："1÷3=0.333...，这个...意思是写不完，但我写3个3就差不多等于1/3了。"这句话有什么问题？0.33和1/3谁大？

2. **找反例**："所有分数除出来都是循环小数。"这句话对吗？举一个反例。

3. **识别陷阱**：计算 5÷6 = ?。小红写 0.8333...，循环节是3。小蓝写 0.83̇。小华写 0.8̇3̇。谁的写法正确？

4. **逆向思考**：一个循环小数是 0.1̇4̇2̇8̇5̇7̇（循环节142857），它是由哪个除法产生的？提示：什么数的倒数的循环节正好是142857？

## 费曼填空 4 道

1. "循环小数之所以会循环，是因为除法中______________________是有限的，当同一个余数再次出现时，后面的计算就______________________。"
2. "除以7时，余数最多有____种可能，所以循环节最多____位。这就像____个人进____个房间，至少有一个房间有两个人。"
3. "循环节是从除法竖式中______________________的位置开始，到______________________的位置结束。"
4. "有限小数是循环小数的特殊情况——它的循环节是______________________，或者说余数在某一步变成了______________________。"

## 分层练习 3 组

**组1 直接应用**：用竖式计算 1÷6、2÷3、5÷11，写出循环小数表示。

**组2 变式迁移**：不用计算，推理 1÷7 的循环节最多几位？1÷7=0.142857142857...，第100位小数是几？如何快速推算？

**组3 综合应用**：比较 0.3̇ 和 0.33，谁大？解释原因。一个分数化成小数是循环小数，循环节是"09"，你能猜出这个分数的分母可能是什么？
