# 组合图形面积 — 教学参考

## 拆到公理

**核心**：组合图形面积基于**面积可加性**——一个图形的总面积等于它各部分面积之和（不重叠的情况下）。解决组合图形 = 把它**分解为已知基本图形** → 分别计算 → 面积求和/求差。

**不可再拆的公理**：
1. 面积可加性：总面积 = 各部分面积之和
2. 基本图形面积公式：矩形（长×宽）、平行四边形（底×高）、三角形（底×高÷2）、梯形（(上底+下底)×高÷2）
3. "分割"和"添补"两种策略等价——都是把未知图形转化为已知图形
4. 分割方式不唯一，但结果必须相同（提供自洽验证）

## 常见误区 Top 3

1. **分割边界重复计算**：把图形切成两块，但中间那条切割线两侧的面积被重复算了。
2. **分割后找错基本图形的高**：在梯形和三角形组合中，找不到或找错对应的高。
3. **"添补"时忘减**：用大矩形包裹不规则形，算了包裹矩形的面积，忘了减掉补上的部分。

## 交互实验室方案

**polygon-cut 组件**：
- 初始：一个 L 形多边形（如 8cm×5cm 矩形缺一个 3cm×2cm 角）
- 拖拽切割线：竖切一刀 → 分成 5×5 正方形 + 5×3 矩形 → 各自计算 → 求和
- 换一种切法：横切一刀 → 分成 8×2 矩形 + 8×3 矩形 → 同样求和
- 添补法模式：用一个 8×5=40 的大矩形包住，减掉 3×2=6 的缺角 → 40-6=34
- 关键：三种方法得到的面积**应该相同**——拖拽查看每种方法的中间计算，让学生看到一致性

## 对抗挑战 4 道

1. **小明的分割**：一个 L 形，小明横着切一刀，上面是 5×2=10，下面是 8×3=24，总面积=34。小华竖着切一刀，左边 5×5=25，右边 3×2=6，总面积=31。两个答案不一样！谁错了？哪一步错了？

2. **找反例**："分割的块数越多越好算。"这句话有什么问题？把一个简单 L 形切成 10 小块，会出现什么麻烦？

3. **识别陷阱**：一个组合图形被分成一个三角形和一个矩形。三角形的底是 4cm，高是 3cm；矩形的长是 4cm，宽是 3cm。小明直接用 4×3 算了两次——他说"三角形底×高就是矩形的面积除以2"。他算对了吗？

4. **逆向思考**：一个组合图形的总面积是 48cm²，它由一个正方形和一个三角形拼成。正方形边长 6cm，三角形的面积是多少？这个三角形可能是什么形状？

## 分层练习 3 组

**组1 直接应用**：一个 L 形，大矩形 10cm×6cm，缺角 4cm×3cm。用分割法和添补法分别计算面积，验证结果相同。

**组2 变式迁移**：一个不规则五边形，把它分成一个矩形 + 两个三角形。矩形 5cm×4cm，三角形A底3cm高2cm，三角形B底4cm高3cm。总面积是多少？

**组3 综合应用**：学校花坛是一个组合图形：半圆形（直径4m）+ 矩形（4m×6m）。π取3.14，花坛面积是多少？如果沿边界围一圈护栏，护栏需要多长？
