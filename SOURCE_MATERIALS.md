## 一、2022版课标 — 第三学段（5-6年级）核心原文

### 学段目标

> 经历用字母表示数的过程，认识自然数的一些特征，**理解小数和分数的意义**；能进行小数和分数的四则运算，**探索数运算的一致性**；形成符号意识、运算能力、推理意识。探索几何图形面积和体积的计算方法，会计算常见平面图形的周长和面积，会计算常见立体图形的体积和表面积；能用有序数对确定点的位置，进一步认识图形的平移、旋转和轴对称；形成量感、空间观念和几何直观。

### 数与运算 — 内容要求

> (1) 知道2,3,5的倍数的特征，了解公倍数和最小公倍数，了解公因数和最大公因数，了解奇数、偶数、质数（素数）和合数。
> (2) 结合具体情境探索并理解小数和分数的意义，**感悟计数单位**；会进行小数、分数的转化，进一步发展数感和符号意识。
> (3) 结合具体情境理解整数除法与分数的关系。
> (4) 能进行简单的小数、分数四则运算和混合运算，**感悟运算的一致性**，发展运算能力和推理意识。

### 数量关系 — 内容要求

> (1) 根据具体情境理解等式的基本性质。
> (3) 在具体情境中，探索用字母表示事物的关系、性质和规律的方法，感悟用字母表示的一般性。

### 图形认识与测量 — 内容要求

> (3) 知道面积单位千米²、公顷；探索并掌握平行四边形、三角形和梯形的面积计算公式；会估计不规则图形的面积。
> (4) 通过实例了解体积（或容积）的意义，知道体积（或容积）的度量单位，**能进行单位之间的换算**。
> (5) 认识长方体、正方体和圆柱，了解这些图形的展开图，探索并掌握这些图形的体积和表面积的计算公式。
> (6) 对于简单物体，**能辨认不同方向（前面、侧面、上面）的形状图**。

### 图形位置与运动 — 内容要求

> (2) 能用有序数对（限于自然数）表示点的位置。
> (4) 能在方格纸上进行简单图形的平移和旋转；认识轴对称图形和对称轴。

## 二、课标教学提示 — 揭示概念依赖关系

### 数与运算教学提示（关键原文）

> **数的认识教学**：在初步认识小数和分数的基础上，引导学生在具体情境中，理解小数和分数的意义，**感悟计数单位**。在教学过程中，可以让学生体验与小数有关的数学文化，理解、描述各数位上数字的意义，进一步提升数感。

→ **对应边**：位置值 → 小数的意义、位置值 → 小数乘法（小数点移位依赖位置值理解）

> **数的运算教学**：应注重对整数、小数和分数四则运算的统筹，让学生进一步**感悟运算的一致性**。例如，在分数加减运算的过程中，引导学生理解**通分的目的是得到同样计数单位**，进一步理解计数单位对分数表达的重要性，理解**整数、分数、小数的加减运算都要在相同计数单位下进行**，感悟加减运算的一致性。

→ **对应边**：整数加减法 → 分数加减法（"计数单位相同才能直接计算"这条铁律）

> 用字母表示的教学要设计合理的实际情境，引导学生会用字母或含有字母的式子表达实际情境中的数量关系、性质和规律。例如：用字母表达常见数量关系及其变形，"路程=速度×时间"表示为s=v×t；**还可以表达图形的周长和面积计算公式等**。

→ **对应边**：简易方程 → 平行四边形面积（related_to，公式中的字母表示）

### 图形与几何教学提示（关键原文）

> **引导学生运用转化的思想，推导平行四边形、三角形、梯形、圆等平面图形的面积公式**，形成空间观念和推理意识。

→ **对应边**：矩形面积 → 平行四边形面积（剪拼→长方形是"转化思想"的起点）

> **引导学生通过对立体图形的测量，从度量的角度认识立体图形的特征**；理解长度、面积、体积都是相应度量单位的累加。

→ **对应边**：整数乘法 → 体积和体积单位（体积=单位立方体个数=长×宽×高，本质是乘法）

> 图形的位置教学可结合教室里学生的位置、电影院里观众的位置等熟悉的情境，引导学生借助方格纸上的点，用有序数对表示具体的位置。

→ **对应边**：（位置在五上，独立单元，与图形与几何交叉）

### 观察物体（课标明确要求）

> 对于简单物体，能辨认不同方向（前面、侧面、上面）的形状图，能把观察的方向与相应形状图对应起来，形成空间观念。

→ **对应边**：图形认知 → 观察物体（先认识平面图形，才能辨认从不同方向投射的形状）
→ **对应边**：观察物体 → 长方体和正方体（多角度观察是认识立体特征的基础）

## 四、课标原文 → 知识图谱边的映射校验

| 边 | 课标原文支撑 |
|----|------------|
| 整数加减法 → 分数加减法 | "整数、分数、小数的加减运算都要在相同计数单位下进行，感悟加减运算的一致性" |
| 位置值 → 小数的意义 | "感悟计数单位"+"理解、描述各数位上数字的意义" |
| 位置值 → 小数乘法 | "感悟计数单位"（小数点移动依赖数位理解） |
| 位置值 → 小数除法 | 同上 |
| 位置值 → 整数四则运算 | "理解、描述各数位上数字的意义"（竖式对齐） |
| 图形认知 → 平行四边形面积 | "引导学生运用转化的思想，推导平行四边形...的面积公式"（必须先认识平行四边形特征） |
| 图形认知 → 观察物体 | "能辨认不同方向的形状图"（前提是认识基本平面图形） |
| 观察物体 → 长方体和正方体 | "能把观察的方向与相应形状图对应起来"→"认识长方体和正方体" |

| 整数乘法 → 单位换算 | "能进行单位之间的换算"（进率×数量） |
| 单位换算 → 体积和体积单位 | "能进行单位之间的换算"+"知道体积的度量单位" |
| 简易方程 → 平行四边形面积 | "还可以表达图形的周长和面积计算公式等" |
