## Codex 读取指南

1. 检查 "当前状态" → 确定要做哪个阶段
2. 参考 `reference/sprint-4/CODEX-TASK.md` 获取详细任务
3. 参考 `reference/sprint-4/batch-X/` 获取每课教学内容
4. 每完成一个阶段 → 提交代码 → 更新本文件 "当前状态"
5. 等待 Ariste 审查 → 审查结论会写入本文件和 `review/` 目录 → 拉取后按反馈修正

## Ariste 读取指南

1. 轮询发现新提交 → 拉取
2. 读取本文件确认当前阶段
3. 审查代码 → 跑审计 → 写 `review/SPRINT-4-<phase>-REVIEW.md`
4. 更新本文件 "当前状态" + "审查记录"
5. 推送 → 群内简报
