🧮 小数 × 整数

乘法的本质没变。变的只是计数单位。

🔑 前置检查
🧱 拆到公理

⚡ 先想3秒：乘法到底是什么？

...
3 × 4 = 4 + 4 + 4 3 × 4 = 3个4相加 所以，乘法的本质是—— 求几个相同加数的和。

追着打：那 3 × 0.4 呢？

还是3个0.4相加。 0.4 + 0.4 + 0.4 = 1.2 3 × 4： 3个1 × 4 = 12个1 3 × 0.4：3个0.1 × 4 = 12个0.1 = 1.2
乘法的本质没变。
变的只是"计数单位"。
学会换单位，小数乘法 = 整数乘法。
"小数乘法就是把小数当整数算，
然后再把计数单位换回来。就这么简单。"

🎯 典例精讲

例1：模式识别 — 买风筝（教材原例）

【示范步骤】 题目：买3个风筝，每个9.5元，需要多少钱？ 第1步 — 换单位：9.5元 = 95角 第2步 — 整数乘：95 × 3 = 285（角） 第3步 — 换回来：285角 = 28.5元 答：需要28.5元。 竖式： 9.5 ← 1位小数 × 3 ───── 28.5 ← 积也有1位小数（先按95×3算，再点小数点）

✏️ 同型自测：买4本练习本，每本2.8元，一共多少钱？

第一步：2.8元 = 角 第二步： × 4 = （角） 第三步： 角 = 元

例2：翻译 — 往返跑步

【示范步骤】 题目：小亮家到学校大约1.3km，他每天往返两次。他每天走多少km？ 第1步 — 读懂"往返两次"： 往返 = 去+回 = 2次单程 往返两次 = 2 × 2 = 4次单程 第2步 — 列式：1.3 × 4 第3步 — 换单位计算： 1.3km × 4 = (13个0.1km) × 4 = 52个0.1km = 5.2km 答：他每天走5.2km。

⚠️ 踩坑预警
🕳️ 坑①：竖式对齐 —— "末位对齐"还是"小数点对齐"？

症状：列竖式时把小数点和乘数对齐（像做加法一样）。

小明（✅正确）： 小红（❌错误）： 9.5 9.5 × 3 × 3 ────── ────── 28.5 285 ← 小数点和3对齐了！

根因：加法要求相同数位对齐→小数点对齐。但乘法是全部相乘再合并。混淆了加法和乘法的竖式规则。

🔗 后续炸雷： 如果这里养成了"乘法小数点对齐"的习惯，到了除法中，列竖式时也会把小数点和被除数对齐，而不是按除数转化。一步错 → 小数除法全错。

🔍 自查法：写完竖式后问自己："我在做加法还是乘法？" 加法→小数点对齐。乘法→末位对齐，之后单独点小数点。

✏️ 纠正练习：判断对错—— 4.6 × 3 = 13.8（竖式末位对齐）✅ 4.6 × 3 = 138（竖式小数点对齐）❌ 忘了点小数点

🕳️ 坑②：小数位数死记硬背

症状：背了"积的小数位数 = 因数小数位数之和"，但遇到0.4×0.5=0.2就慌了——"一位+一位=两位，0.2只有一位？我算错了？"（其实0.20=0.2）

根因：口诀正确，但学生不理解为什么，变成了机械口诀。没有内化"每次乘法的单位变化"。

🔗 后续炸雷： 死记口诀到分数乘法彻底崩盘。到五年级下学2/3×1/4，背"分子×分子、分母×分母"，而不理解"2/3的一半=1/3"。口决算不出理解。

🔍 自查法：每次做完小数乘法，用换单位法重做一次验证： 0.4×0.5 → (4个0.1)×(5个0.1) → 20个(0.1×0.1) → 20个0.01 → 0.20=0.2 ✓ 换单位法算出的答案如果和口诀不一样 → 优先相信换单位法。
