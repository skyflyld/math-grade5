🏠›分数加减法

🧮 分数加减法 — 同单位才能合并

分数加减和整数加减一样：先对齐单位，再改变数量。

分母不是要相加的数，它是单位

3 个苹果加 2 个苹果，结果是 5 个苹果，不是 5 个梨。分数也是一样：只有分数单位相同，分子才可以直接加减。

分数单位通分约分
1/4 + 1/6 3/12 + 2/12 = 5/12
🔑预热
🧱拆到公理
公理 1

只有相同单位才能直接加减

整数加减要个位对个位、十位对十位；分数加减要分母相同，也就是每一份大小相同。

同分母

单位相同，只改变取了几份。3/7 + 2/7 = 5/7。

异分母

单位不同，先通分。1/4 + 1/6 = 3/12 + 2/12。

结果约分

算完后检查能不能写得更简洁，例如 3/6 = 1/2。

公理 2

加减的是“份数”，不是“单位名”

1/3 + 1/3 = 2/3。分母 3 表示单位是“三分之一”，两个三分之一合起来还是用三分之一作单位。

错法 1/3 + 1/3 = 2/6，相当于把“苹果”也加成了另一个单位。
分数加减 = 统一单位 → 加减份数 → 约成最简
🎮交互实验室 · 通分到加减再约简

看分数条时，只问一个问题：两条刻度是不是同一把尺子？是，才能加减；不是，先通分。

📐教材原文
📖 教材：“同分母分数相加，分母不变，分子相加。异分母分数相加，先通分。计算结果能约分的要约分。” （点击看拆解）
拆解：分母不变，因为单位不变；异分母先通分，因为单位不同；最后约分，是把同一个值写得更简洁。
🎯典例精讲

例 1：3/7 + 2/7

单位都是 1/7，只加份数：3+2=5，所以结果是 5/7。

例 2：1/4 + 1/6

最小公分母 12：1/4=3/12，1/6=2/12，所以结果是 5/12。

例 3：5/8 - 1/3

通分到 24：15/24 - 8/24 = 7/24。

例 4：2/6 + 1/6

先算 3/6，再约分为 1/2。3/6 对，但不是最简。

⚠️踩坑预警
🕳️ 分母也相加

症状：1/3 + 1/3 = 2/6。

根因：不知道分母是单位大小，不是要合并的数量。

正确链路：两个 1/3 合起来是 2 个 1/3，所以是 2/3。
🕳️ 只通分不约分

症状：2/6 + 1/6 = 3/6 就停下。

根因：把“算出一个等值答案”和“写成最简答案”混为一谈。
